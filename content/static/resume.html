Work Experience

  • 2013 - Present Senior Software Engineer at Google

    Chrome Media User Experience (2015 - Present)
    Lead for the Chrome Media User Experience effort. Drove roadmap and priorities for a team of 4+ engineers across offices. Launched features such as Media Session API, muted autoplay, Remote Playback API, Presentation API, etc.

    Progressive Web Apps (2013 - 2015)
    Design and implementation of Web APIs in the context of Progressive Web Apps: Screen Orientation API, Web Manifest, Permissions API. Contributed to Push Notification and Service Worker.

  • 2010 - 2013 Senior Platform Engineer at Mozilla

    Designed and implemented new APIs for the Web Platform to support the Firefox OS project and make the Web a competitive platform on Mobile. Involved in most API designs for Firefox OS, I have implemented a few of them in Firefox for Android and pushed some of them to standardization bodies. Also took part of the design of the security model of Firefox OS and implemented key parts of it.

    As a reviewer for the HAL and HTML/DOM modules and a Mozilla super-reviewer, I do a lot of code reviews to approve patches to land in mozilla-central and super-reviews to validate API changes.

    Involved in the W3C specification process as a co-chair of the SysApps working group and an active member of the WHATWG, HTML, DAP and WebApps working groups. I am editor of the Battery Status API, the Screen Orientation API, the Runtime and Security Model for Web Applications and the Network Information API.

    Implemented HTML5 Forms in Firefox.

  • 2009 Google Summer of Code
    Google Summer of Code is a Google sponsored 3-month contribution to a FLOSS project.

    Implemented a Gentoo Portage backend for PackageKit, an abstraction layer on top of package managers for Linux distributions. In addition of implementing the backend, this work involved extending Portage APIs, fixing bugs and adding features to Portage and PackageKit.

  • 2007 - 2009 Gameplay Programmer/Platform Engineer at Ubisoft

    While at university, I did a few internships and short term contracts at Ubisoft, working on various games at various positions.

    Rayman Raving Rabbids 2 (Nintendo Wii) - 8 months
    I joined the team at the beginning of the project and left to get back to school after the code freeze. I started by prototyping some mini-games, and then was responsible for a regular mini-game and all six FPS mini-games. As the only Gameplay Programmer on these, I wrote the code handling the game mechanisms such as AI code to handle all sorts of enemies.

    Ghost Recon: Future Soldier (XBox 360/PS3) - 5 months
    I improved re-usability, maintenance and quality of a core part of a 10-year old game engine written in C/C++ with a short-term goal of reducing memory usage. I started an effort to have the code covered by a test suite.

    Undisclosed project - 3 months
    I was responsible for prototyping gameplay ideas for an experimental music-related game. This task required a lot of interactionis with the Game Designer and the ability to quickly implement ideas and improve them through try/fix iterations.

  • 2006 Platform Engineer at Dancing Dots Studio (5-month internship)

    I wrote a 2D/3D sound engine based on OpenAL and a GUI engine based on CEGUI for the PC game Ride!. I also wrote the prototype of the rhythm game mode.
