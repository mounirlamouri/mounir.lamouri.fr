  • 2009 Google Summer of Code

    REDUCE THE SIZE
    Implementation of a Gentoo Portage backend for PackageKit.

    PackageKit is an abstraction layer on top of package managers for Linux distributions. The goal being to simplify package installation and management by allowing applications to query package status or request package installations from an unified API. It also allows a unified cross-distribution GUI to manage packages.

    The goal of the Summer of Code was to implement a backend for Portage, the main and historical package manager for Gentoo written in Python. This work involved understanding, and extending Portage external APIs but also fixing some Portage bugs and writing new features.

    The principal challenge of this project was to get involved in two communities at the same time, having to deal with two different APIs written in different languages (C for PackageKit and Python for Portage) and write patches for both of them.
